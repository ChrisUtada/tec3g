# 配方与探索配置手册

> 所有配方和探索场景配置通过代码构建，见 `autoload/recipe_registry.gd` 和 `autoload/scene_config_registry.gd`。

---

## 堆叠配方 — RecipeRegistry

**文件：** `autoload/recipe_registry.gd`

### 注册新配方

在 `_ready()` 中，找到对应卡牌的 `_register()` 调用，添加 `_recipe()` 行：

```gdscript
_register("ITEM_sdt", [
    _recipe([TARGET...], RESULT, MIN, MAX, WEIGHT, DROPS, "LABEL"),
])
```

### _recipe 参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `targets` | `Array[CardData]` | 堆叠链中需要存在的卡牌（用 `[]` 包裹，多个卡牌表示都需要存在） |
| `result` | `CardData` | 合成产出的卡牌 |
| `min_c` | int | 最少产出几张（1~100） |
| `max_c` | int | 最多产出几张（1~100） |
| `w` | float | 概率权重（多个配方竞争时按权重随机） |
| `drops` | int | 全局限定掉落次数（0=不限） |
| `label` | String | 日志/调试标识 |

### 示例

```gdscript
# 单目标：手电筒 + 观测 → 1~2 窥视真相
_recipe([preload("res://resources/cards/LOGIC_observe.tres")],
        preload("res://resources/cards/ITEM_peek_truth.tres"), 1, 2, 1.0, 0, "窥视真相")

# 多目标：CardA + CardB + CardC → 结果
_recipe([preload("res://resources/cards/CardA.tres"),
         preload("res://resources/cards/CardB.tres"),
         preload("res://resources/cards/CardC.tres")],
        preload("res://resources/cards/ITEM_result.tres"), 1, 1, 1.0, 0, "三卡合成")
```

## 探索场景配置 — SceneConfigRegistry

**文件：** `autoload/scene_config_registry.gd`

### 注册新场景

在 `_ready()` 中添加一行 `_configs["SCENE_xxx"] = _xxx()`，然后新建 `func _xxx() -> ExplorationConfig` 函数。

### 辅助函数

#### _slot(ids, hint) — 创建槽位配置

| 参数 | 类型 | 说明 |
|------|------|------|
| `ids` | `Array[String]` | 接受哪些卡牌 ID（如 `["ITEM_sdt"]`） |
| `hint` | String | 提示文字 |

#### _drop(card, min_c, max_c, w, drops, stackable, label) — 创建掉落配方

| 参数 | 类型 | 说明 |
|------|------|------|
| `card` | `CardData` | 掉落哪张卡 |
| `min_c` | int | 最少掉落几张 |
| `max_c` | int | 最多掉落几张 |
| `w` | float | 概率权重 |
| `drops` | int | 全局限定次数（0=不限） |
| `stackable` | bool | `false`=场上已有同名卡则跳过；`true`=不受限制 |
| `label` | String | 日志标识 |

### 完整示例

```gdscript
func _plant_hunter() -> ExplorationConfig:
    var c = ExplorationConfig.new()
    c.scene_card_id = "SCENE_plant_hunter"
    c.scene_name = "植物学家的花园"
    c.scene_description = "描述文字"
    c.slot_count = 3
    c.explore_duration = 3.0

    var branch = SlotBranchRecipe.new()
    branch.branch_name = "采集"
    branch.required_cards = [
        _slot(["ITEM_sdt"], "需要手电筒"),
        _slot(["ITEM_coin"], "需要因果律"),
    ] as Array[PanelSlotConfig]
    branch.result_recipes = [
        _drop(preload("res://resources/cards/ITEM_plant.tres"), 1, 1, 1.0, 1, false, "采集植物"),
    ] as Array[DropRecipe]
    c.branch_recipes.append(branch)

    return c
```

## 配置 vs 代码对比

| 操作 | 旧方式（已废弃） | 新方式 |
|------|-----------------|--------|
| 改配方 | 打开 .tres → 展开 sub_resource → 改字段 | 打开 `recipe_registry.gd` → 改 `_recipe()` 参数 |
| 新增配方 | 编辑器 ➕ 新增 → 层层设字段 | 加一行 `_recipe()` |
| 改探索掉落 | 打开场景 .tres → 展开 exp_conf → 展开 branch → 展开 result_recipes | 打开 `scene_config_registry.gd` → 改 `_drop()` 参数 |
| 新增场景分支 | 编辑器操作十分钟 | 加 5 行代码 |
